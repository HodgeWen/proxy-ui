# Requirements Archive: v1.0 MVP

**Archived:** 2026-02-19
**Core Value:** 提供一个界面美观、体验流畅、部署简单的 sing-box 管理面板

## v1 Requirements (42/42 Satisfied)

### 入站管理 (Inbound) — 10/10

- [x] **INB-01**: 用户可以通过弹框表单添加 VLESS 入站配置
- [x] **INB-02**: 用户可以通过弹框表单添加 Hysteria2 入站配置
- [x] **INB-03**: 用户可以编辑已有的入站配置
- [x] **INB-04**: 用户可以删除入站配置
- [x] **INB-05**: 用户可以配置入站的端口、监听地址、标签
- [x] **INB-06**: 用户可以配置入站的 TLS 选项（启用/禁用、关联证书）
- [x] **INB-07**: 用户可以配置高级传输选项（WebSocket, gRPC, HTTP/2）
- [x] **INB-08**: 表单字段使用智能默认值预填（flow、transport、端口等最佳实践）
- [x] **INB-09**: 每个表单字段旁有 info 图标，悬浮显示该字段的用途说明
- [x] **INB-10**: 配置变更后自动触发 `sing-box check` 验证，通过后应用并重启

### 用户管理 (User) — 9/9

- [x] **USR-01**: 管理员可以通过弹框表单创建用户（用户名、备注）
- [x] **USR-02**: 管理员可以编辑用户信息
- [x] **USR-03**: 管理员可以删除用户
- [x] **USR-04**: 管理员可以将用户分配到一个或多个入站节点
- [x] **USR-05**: 管理员可以为用户设置流量上限（字节数）
- [x] **USR-06**: 管理员可以为用户设置到期时间
- [x] **USR-07**: 系统自动为 VLESS 用户生成 UUID，支持一键复制
- [x] **USR-08**: 管理员可以通过关键字搜索和筛选用户列表
- [x] **USR-09**: 管理员可以批量选择用户执行删除、启用/禁用、流量重置操作

### 订阅系统 (Subscription) — 5/5

- [x] **SUB-01**: 系统为每个用户生成唯一的订阅链接
- [x] **SUB-02**: 订阅链接返回 Base64 编码的代理列表（V2ray 标准格式）
- [x] **SUB-03**: 订阅链接支持 Clash/ClashMeta YAML 格式输出
- [x] **SUB-04**: 用户可以为每个节点生成 QR 码（支持扫码导入客户端）
- [x] **SUB-05**: 订阅链接附带信息页，显示用户用量、到期时间、节点列表

### 证书管理 (Certificate) — 2/2

- [x] **CRT-01**: 管理员可以手动指定 TLS 证书路径（fullchain + privkey）
- [x] **CRT-02**: 管理员可以将证书关联到入站配置的 TLS 选项

### 流量统计 (Traffic) — 2/2

- [x] **TRF-01**: 系统记录并展示每个用户的上行/下行流量使用量
- [x] **TRF-02**: 系统记录并展示每个入站节点的总流量使用量

### sing-box 核心管理 (Core) — 4/4

- [x] **COR-01**: 面板可以应用配置变更并重启 sing-box 进程
- [x] **COR-02**: 面板启动时自动检测 sing-box 版本并显示
- [x] **COR-03**: 管理员可以在面板中将 sing-box 核心更新到最新版本
- [x] **COR-04**: 管理员可以将 sing-box 核心回滚到上一个已安装版本

### 面板系统 (System) — 5/5

- [x] **SYS-01**: 管理员通过用户名/密码登录面板（首次部署必须设置，无默认密码）
- [x] **SYS-02**: 面板支持 HTTPS 访问
- [x] **SYS-03**: 支持 Docker Compose 一条命令部署
- [x] **SYS-04**: 支持 bash 安装脚本部署（curl | bash）
- [x] **SYS-05**: 支持下载单二进制文件 + 配置文件运行

### 界面体验 (UX) — 5/5

- [x] **UX-01**: 暗色主题 UI（Vercel/Linear 风格）
- [x] **UX-02**: 使用 shadcn/ui 组件库 + Tailwind CSS
- [x] **UX-03**: 使用 react-bits 库增强微交互 — ⚠️ PARTIAL（已安装未使用，shadcn 过渡足够）
- [x] **UX-04**: 界面语言为中文（专业术语如 VLESS、TLS、Hysteria2 保留英文）
- [x] **UX-05**: 所有配置操作通过弹框（Modal）表单完成，不跳转页面
